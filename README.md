# 📈 Stock Market Price Prediction App

This Streamlit app predicts stock prices using historical data and a Random Forest Regressor model.

## 🔍 Features
- Load historical stock data (AAPL, etc.)
- Predict next day's closing price
- Visualize stock trends
- Built with Streamlit + Scikit-learn

## 📁 Project Structure
stock_market_price_prediction/
├── app.py
├── models/
├── data/
├── notebooks/
├── src/
├── requirements.txt

## 🚀 How to Run

1. Clone the repo  
2. Install dependencies  
3. Run the app:

```bash
streamlit run app.py
